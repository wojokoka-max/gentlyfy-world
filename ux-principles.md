# Gentlyfy UX Principles

## Core Philosophy

The platform should regulate, not overstimulate.

Users should feel:
- calmer,
- safer,
- slower,
- emotionally grounded.

## Never Use

- endless feeds,
- autoplay noise,
- aggressive notifications,
- engagement pressure,
- addictive mechanics,
- productivity guilt,
- overstimulating animations,
- algorithmic chaos.

## Always Prioritize

- emotional pacing,
- silence,
- softness,
- sensory safety,
- curiosity,
- gentle discovery,
- calm transitions,
- intentional interaction.

## UX Emotional Goal

The user should think:

"I wonder what I’ll discover today."
